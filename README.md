## FSD

App:
настройки приложения, глобальные провайдеры, роутинг, стили

Pages:
страницы приложения с конкретным UI, маршрутами и контейнерами

Widgets:
крупные блоки интерфейса, собранные из фич и сущностей

Features:
отдельные пользовательские фичи, возможности (например, «добавление товара в корзину», «авторизация»)

Entities:
модули с описанием и логикой «сущностей» предметной области (User, Todo, Product, Order и т.д.)

Shared:
общие библиотеки, утилиты, переиспользуемые компоненты (кнопки, иконки, функции валидации)
